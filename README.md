# Precision Agriculture and Resource Optimization

## Short Project Overview

This project is an AI-driven precision agriculture decision-support system. It uses soil nutrient and environmental data to support crop recommendation and resource optimization for practical farming decisions.

## Dataset

This project uses the Kaggle Crop Recommendation Dataset:
https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset

Dataset features:

- N
- P
- K
- temperature
- humidity
- ph
- rainfall
- label

## Project Workstreams

The project has two connected parts:

**Project 1: Intelligent Crop Recommendation**

- Answers what crop should be planted.

**Project 2: Fertilizer Optimization Intelligence**

- Answers how nutrients, soil conditions, and resources should be managed to support the selected crop.

## Current Goal

The goal is to build a graduate-level machine learning project that is practical, realistic, and not overengineered.
